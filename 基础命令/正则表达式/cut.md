```bash
cut [选项] 文件名
```

### 选项

| 选项 | 备注                        |
| ---- | --------------------------- |
| -f   | 列号 提取第几列             |
| -d   | 分隔符 按照指定分隔符分割列 |

e.g.

```bash
cut -f 2 a.txt
cut -f 2,4 a.txt
cut -d ":" -f 1,3 /etc/passwd
```

## cut 命令的局限

```bash
df -h | cut -d " " -f 1,3
# 空格无法使用
```
