| 多命令执行符 | 格式               | 作用   |
| ------------ | ------------------ | ------ |
| ;            | 命令 1 ; 命令 2    |        |
| &&           | 命令 1 && 命令 2   | 逻辑与 |
| \|\|         | 命令 1 \|\| 命令 2 | 逻辑或 |

```bash
ls && echo yes || echo no
```

## 管道符

```bash
命令 1 | 命令 2
# 命令1的正确输出作为命令2的操作对象
```

e.g.

```bash
ls -l /etc/ | more
netstat -an | grep ESTABLISHED
```
