| 通配符 | 备注                         |
| ------ | ---------------------------- |
| \*     | 匹配任意内容                 |
| ?      | 匹配任意一个字符             |
| []     | 匹配任意一个中括号内的字符   |
| [-]    | 匹配中括号范围内任意一个字符 |
| [^]    | 匹配不是中括号内的一个字符   |

```bash
a=123
echo $a
# 123
echo '$a'
# $a
echo "$a"
# 123
```

## bash 中的其他特殊符号

| 符号 | 作用                         |
| ---- | ---------------------------- |
| ''   | 单引号                       |
| ''   | 双引号，$，\\，\` 有特殊含义 |
| \`\` | 反引号，调用系统命令         |
| $()  | 同反引号                     |
| #    | 注释                         |
| $    | 调用变量的值                 |
| \    | 转义符                       |
