## 测试选项

| 测试选项          | 作用                                                                                                              |
| ----------------- | ----------------------------------------------------------------------------------------------------------------- |
| 文件 1 -nt 文件 2 | 判断该文件 1 的修改时间是否比文件 2 的新（如果新则为真）                                                          |
| 文件 1 -ot 文件 2 | 判断该文件 1 的修改时间是否比文件 2 的旧（如果旧则为真）                                                          |
| 文件 1 -ef 文件 2 | 判断该文件 1 是否和文件 2 的 inode 号一致，可以理解为两个文件是否为同一个文件。这个判断用于判断硬链接是很好的方法 |

e.g.

```bash
ln /root/a.txt /tmp/a
# 创建个硬链接
[ /root/a.txt -ef /tmp/a ] && echo "yes" || echo "no"
# 返回yes
```
