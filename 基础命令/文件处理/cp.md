```bash
cp [选项] [原文件或目录] [目标目录]
```

- copy

### 选项

| 选项 | 备注                               |
| ---- | ---------------------------------- |
| -r   | 复制目录                           |
| -p   | 连带文件属性复制                   |
| -d   | 若源文件是链接文件，则复制链接属性 |
| -a   | 相当于 -pdr                        |

e.g.

```bash
cp test.log test.log.bak
```
