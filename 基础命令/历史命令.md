```bash
history [选项] [历史命令保存文件]
```

### 选项

| 选项 | 备注                                                  |
| ---- | ----------------------------------------------------- |
| -c   | 清空历史命令                                          |
| -w   | 把缓存中的历史命令写入历史命令保存文件~/.bash_history |

```bash
/etc/profile
# 设置HISTSIZE
```

## 历史命令的调用

```bash
# 使用上下箭头调用以前的历史命令
!100
# 使用"!n"重复执行第n条历史命令
!!
# 使用"!!"重复执行上一条命令
!hist
# 使用"!字串"重复执行最后一条以该字串开头的命令
```
