### 运行级别

| 运行级别 | 含义                                                        |
| -------- | ----------------------------------------------------------- |
| 0        | 关机                                                        |
| 1        | 单用户模式，可以想象为 windows 的安全模式，主要用于系统修复 |
| 2        | 不完全的命令行模式，不含 NFS 服务                           |
| 3        | 完全的命令行模式，就是标准字符界面                          |
| 4        | 系统保留                                                    |
| 5        | 图形模式                                                    |
| 6        | 重启动                                                      |

```bash
runlevel
# 查看运行级别命令
# 进入当前级别之前的级别 当前级别
init 运行级别
# 修改运行级别命令
vim /etc/inittab
# 修改系统默认运行级别
# id:3:initdefault:
# 系统开机后直接进入哪个运行级别
```
